
# **NBA Players Stats Analysis**

---

## 📌 Project Overview

Basketball debates often hinge on subjective opinions, especially when it comes to the GOAT conversation. This project brings data to the forefront. Using MySQL, I analyzed over 12,000 player-season records spanning 27 NBA seasons (1996 - 2022) to uncover performance trends, compare eras, and identify statistically driven MVPs.

## 🎯 Objectives

- To identify the most excellent players through all seasons in scoring, rebounding, and playmaking. 
- To examine shifts in player size, playing style, and athletic performance across historical eras.  
- To identify the teams which consistently produce the top players.

  
## 🛠️ Tools Used

- Python: Pandas, SQLAlchemy, pymysql
- MySQL Workbench  

## 🧠 Skills Demonstrated

- SQL analysis using CTEs, subqueries, joins, and aggregations    
---

### 📥 Data Loading  
Raw data was imported into a Jupyter notebook using `pandas`, check the basic situation of data, then exported to MySQL Workbench via `SQLAlchemy` and `pymysql`.

- In 1990s the player average height is 200.6 centimeters, and the average weight is 100.01 kg;
- In 1990s the player average height is 198.82 centimeters, and the average weight is 97.78 kg.

- The teams which consistently produce top players are : MIA(11 players), CLE(8 players), MEM(8 players), POR(7 players), CHI(6 players).

- The rookies' contributions are : average points 5.68, average rebounds 2.59, average assists 1.2;
- The veteran's contributions are : average points 8.48, average rebounds 3.66, average assists 1.89.
